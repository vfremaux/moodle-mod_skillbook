Tâches

Les tâches sont des réalisations ou des activités proposées par les enseignants aux étudiants de leur cours. Les tâches sont en quelque sorte le pendant enseignant des activités des étudiants.

Seuls les enseignant peuvent proposer des tâches. Seuls les étudiants peuvent souscrire à des tâches.

Composition d'une tâche

  • Une tâche a un auteur (enseignant) ;
  • un titre ;
  • une description de la mission à remplir (cahier des charges) ;
  • une liste de recommandations ;
  • une liste de compétences du référentiel associé ;
  • une date de début et une date de fin ;
  • des documents associés dit "de consignes".

Le mécanisme de validation d'une tâche est propagé aux compétences pointées par celle-ci. De la sorte les compétences associées à une tâche sont comptabilisées pour la certification des étudiants souscripteurs.

Créer une nouvelle tâche

Seuls les enseignants peuvent créer des tâches

  • Type / thème / titre de la tâche : En typant une tâche vous facilitez son repérage par les étudiants.
  • Description : Indiquez ici le contexte et les objectifs de la tâche.
  • Compétences mobilisées par cette tâche : Celles que l'accomplissement de la tâche légitime.
  • Consignes : Vous pouvez énumérer les critères de réussite afin de faciliter la réalisation de la tâche.
  • Documents attachés : Vous pouvez joindre des documents pour illustrer la tâche ou la cadrer.
    Cocher 'Oui' si vous souhaitez attacher un document.
    • Description du document : Une courte fiche descriptive
    • Type de document [Texte, Exe, PDF, ZIP, Image, Audio, Video, etc.] : Afin d'en faciliter l'affichage.
    • URL : Copiez l'adresse Web du document (ce peut aussi être le lien vers un document déposé dans l'espace Moodle ou vers une activité Moodle...)

Souscription à une tâche

La souscription à une tâche consiste pour un étudiant à cocher celle-ci dans la liste des tâches (onglet "Tâche"). Ce faisant une nouvelle activité lui est automatiquement ajoutée, portant sur le contenu de la tâche souscrite (onglet "Activité").
Il y a différentes modalités de souscription :

  • Souscription libre : chacun décide ou pas de souscrire à une tâche ;
  • Souscription restreinte à un groupe : l'enseignant désigne les étudiants susceptibles de souscrire librement à une tâche ; les autres ne peuvent y souscrire ;
  • Souscription imposée : l'enseignant inscrit d'office un groupe d'étudiants à la tâche.
Ces modalités peuvent bien sûr se combiner entre elles...

Validation d'une tâche

Seuls les enseignants peuvent valider des tâches.
Il y a deux façons de procéder :

  • Soit valider individuellement les activités qui ont été créées lors de la souscription à la tâche ;
  • Soit valider globalement une tâche (en cochant celle-ci dans l'onglet "Tâche") ; dans ce cas toutes les activités créées lors de la souscription sont validées sans vérification par l'enseignant.

Exportation / importation

Il est possible d'exporter des tâches d'un cours pour les importer dans un autre cours.
