Exporter un référentiel

En exportant un référentiel au format XML vous produisez une sauvegare que vous pouvez restaurer dans votre cours Moodle par la fonction symétrique Importer).
Vous pouvez aussi utiliser le format CSV qui est un format textuel de données tabulaires (comme dans un tableur). Le format CSV utilise le séparareur ';' entre les cellules de la table, il faut donc éviter d'utiliser ce symbole dans les textes à enregistrer.

En exportant un référentiel au format XHTML vous produisez un fichier que vous pouvez afficher comme une page Web.

Les fichiers exportés par le module référentiel sont placés dans le répertoire "referentiel" du dossier des données du cours (à chercher dans le dossier MoodleData du serveur Moodle).
