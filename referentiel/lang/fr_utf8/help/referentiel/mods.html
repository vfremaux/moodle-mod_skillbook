Instance du référentiel

Une instance du référentiel est une activité au sens Moodle (comme le sont "Devoir", "Forum" ou "Leçon") s'appuyant sur un référentiel et qui permet aux élèves / étudiants :

- de déclarer des activités mobilisant les compétences du référentiel
- de pointer ces compétences

et qui permet aux enseignants d'évaluer ces compétences...

Lors de la création d'une nouvelle instance (il peut en cohabiter plusieurs, dans le même cours, soit pointant vers le même référentiel, soit vers des référentiels différents) il faut définir :

  • Titre de l'instance [obligatoire] : C'est le nom sous lequel cette instance du référentiel apparaît comme activité intégrée à un cours...
  • Description de l'instance [facultatif] : Courte notice destinée à éclairer les élèves / étudiants sur les objectifs de cette activité.
  • Comment faut-il dénommer un domaine ? [facultatif] : Pour les affichages, sous quel terme est renommé le concept de "domaine".
  • Comment faut-il dénommer une compétence ? : Pour les affichages, sous quel terme est renommé le concept de "compétence".
  • Comment faut-il dénommer un item ? : Pour les affichages, sous quel terme est renommé le concept d'"item".
Vous pouvez aussi définir les paramètres de configuration et d'impression au niveau de l'instance, en fonction des paramètres définis au niveau du site par l'administrateur
  • Configuration : Accès aux données de scolarité et autorisations concernant l'import, la modification et l'impression des certificats.
  • Impression des certificats : Définir quelles informations sont imprimées dans les certificats.
