Exporter les activités

Pour exporter les activités d'une instance du référentiel vous avez le choix entre plusieurs formats :

  • Le format XML destiné à la sauvegarde en vue d'une restauration ultérieure (cette fonction n'est pas implantée actuellement)
  • Le format CSV est un format textuel de données tabulaires (comme dans un tableur).
    Le format CSV utilise le séparareur ';' entre les cellules de la table, il faut donc éviter d'utiliser ce symbole dans les textes à enregistrer.

  • Les formats XHTML et HTML génèrent des fichiers qui s'affichent comme une page Web.

Les fichiers exportés par le module référentiel sont placés dans le répertoire "referentiel" du dossier des données du cours (à chercher dans le dossier MoodleData du serveur Moodle).
