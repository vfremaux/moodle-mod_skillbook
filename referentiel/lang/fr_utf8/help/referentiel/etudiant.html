Profils

Selon le type de référentiel diverses informations doivent être disponibles pour émettre les certificats

Enregistrez les dans les fiches étudiant et établissement.

Elèves

Une fiche étudiant contient :

  • Le numéro d'étudiant ;
    N.B. Par défaut le numéro d'étudiant prend la valeur du 'idnumber' de la table 'user' de Moodle.
    Si ce champ est vide c'est la valeur du champ 'username' (le login) qui est retenue.
    Vous pouvez à tout moment modifier le contenu de ce champ.
  • Le lieu de naissance et le département de naissance ;
  • L'adresse postale de correspondance ;
  • La référence de l'établissement ;

Les étudiants peuvent modifier leur propre fiche.

Etablissements

Une fiche établissement contient :

  • Le numéro d'établissement ;
  • Le nom d'établissement ;
  • La référence de l'établissement ;
  • L'adresse de l'établissement;

Seuls les enseignants avec droit d'édition peuvent modifier les fiches d'établissement.
