Exportation des certificats

Données exportées

L'export des certificats est destiné à l'archivage ou au transfert vers des services de scolarité.
Deux formats sont disponibles

  1. Format complet : toutes les données sont exportées
  2. Format condensé : l'identifiant, le nom, le prénom et les pourcentages de compétences validées sont exportés, ce qui suffit en général pour les services de scolarité.
Pour sélectionner les données à traiter il convient plutôt de passer par l'onglet "Imprimer".

Rôles

Les administrateurs peuvent exporter tous les certificats correspondant à un référentiel donné.
Les enseignants avec droit d'édition ne peuvent exporter que les certificats d'étudiants du cours.

Formats d'exportation des certificats

  • XML est un format textuel de données représentant l'information comme une hiérarchies de balises et de données imbriquées. Il permet la sauvegarde et la restauration de référentiels, d'activités et de certificats.

  • XHTML est un format d'exportation de type page Web destiné à l'affichage.

  • CSV est un format textuel de données tabulaires avec séparateur ';'

    Le format CSV est mal adapté aux données imbriquées comme un référentiel de compétences. Il permet toutefois d'utiliser un tableur comme éditeur ce qui le rend particulièrement pratique.
