Importer / Exporter un référentiel sous forme de barème d'objectifs

Si les objectifs sont activés sur votre serveur Moodle (voir ci après comment les activer) vous pouvez sauvegarder le référentiel sous forme d'un barème d'objectifs puis utiliser ce barème pour évaluer toute forme d'activité Moodle (forums, devoirs, bases de données, wiki, etc.)
Le module Référentiel récupèrera ces évaluations et génèrera des déclarations qui seront dès lors accessibles dans la liste des activités du module référentiel.

Protocole

Le protocole est en trois étapes :

  1. Activer les objectifs au niveau du site
  2. Importer un fichier de barème par objectif
  3. Créer des activités et sélectionner ce barème pour les noter

Activer les objectifs au niveau du serveur

Avec le rôle d'administrateur activez les Objectifs au niveau du serveur (Rubrique "Administration du site / Notes / Réglages généraux :: Activer les objectifs = 'Oui'")
Vous pouvez alors importer un fichier d'objectifs du module Référentiel en passant par le rubrique "Administration du site / Notes / Objectifs / Importer des objectifs.

Créer un fichier d'objectifs depuis le module Référentiel

Avec un rôle d'enseignant passer par l'onglet "Référentiel / Exporter" puis Barème et Objectifs et cliquer sur le lien "Exporter le référentiel sous forme de barème et d'objectifs".
Le fichier "outcomes_referentiel_xxx.csv" sera enregistré sur votre disque dur.

Intéger des objectifs dans votre cours

Il y a deux situations possibles :

  • Soit le barème d'objectifs est disponible au niveau du site, comme liste d'objectifs standards ; ajoutez les tous à votre cours en passant par la rubrique "Administration / Objectifs / Objectifs utilisés dans le cours"
  • Soit le barème n'est pas encore disponible et il faut le charger en passant par "Administration / Objectifs / Modifier les objectifs"
    1. Choisir alors Importer comme objectifs de ce cours ou Importer comme objectifs standards
    2. puis Importer des objectifs (Taille maximale : xx Mo), sélectionner le fichier "outcomes_referentiel_xxx.csv" ci-dessus enregistré.

Mise en oeuvre

Désormais vous pouvez utiliser ce barème d'objectifs pour évaluer toute activité du cours.
Les étudiants notés selon ce barème verront leurs productions intégrées directement dans le module référentiel sous forme de déclarations d'activité accessibles et modifiables selon les modalités usuelles.
N.B. : Cette intégration est liée au lancement périodique du programme cron sur votre serveur.
