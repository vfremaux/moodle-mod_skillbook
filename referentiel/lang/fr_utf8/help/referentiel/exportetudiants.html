Exporter / Importer des données de scolarité

Les données de scolarité sont indispensables pour l'impression des certificats
N.B. Seuls les enseignants avec droit d'édition peuvent réaliser ces opérations à condition que l'administrateur du site Moodle les y autorise en paramétrant le module Référentiel de façon ad hoc.
Afin d'importer des données de scolarité dans le module Référentiel de Moodle suivez scrupuleusement la procédure ci-après faute de quoi l'opération échouera.

La procédure d'exportation est l'étape préalable et obligée qui permet de préparer un fichier sous éditeur (par exemple un tableur) afin de pouvoir ensuite importer ces données de scolarité dans Moodle.
En effet les données de scolarité que l'on importe dans Moodle (par l'onglet Importer) doivent contenir l'identifiant attribué par Moodle à chaque utilisateur lors de la création des comptes dans Moodle.

  1. Première étape
    obtenir une sauvegarde contenant les identifiants Moodle des établissements et des étudiants.
    Commencez par mettre à jour la liste des établissements du cours (onglet Etablissements)
    Puis affichez la liste de tous les étudiants du cours (onglet Etudiants).
    N.B. : Inutile de compléter les champs manquants, vous le ferez hors de Moodle sous éditeur.
    Exportez ensuite (onglet Exporter) les données de scolarité de votre cours au format CSV.
    Le fichier exporté se trouve dans le dossier moddata/referentiel de votre cours Moodle.
  2. Seconde étape
    Modifier cette sauvegarde avec un éditeur (un tableur). Tous les champs marqués 'INCONNU' peuvent être modifiés sous éditeur
    Attention : Champs à ne modifier sous aucun prétexte :
    • #id_etablissement
    • #id_etudiant
    • user_id
    • login

    N.B : Il suffit qu'un des trois champs '#id_etudiant', 'user_id' ou 'login' soit correctement renseigné pour chaque étudiant.
  3. Troisème étape
    Importer ce fichier de scolarité désormais à jour (onglet Importer, format CSV avec séparateur ';' )
  4. Dernière étape
    Exporter les certificats (onglet Certificat), soit au format PDF pour impression, soit au format CSV pour injection dans un logiciel de scolarité type Apogée
