Export of certificates

>

Exported data

The export of certificates is intended for the filing or for the transfer towards services of schooling.
Two sizes(formats) are available

  1. Complete size(format): all the data are exported
  2. Condensed size(format): the identifier, the name, the first name and the percentages of validated skills are exported, what is generally enough for the services of schooling.

To select the data to be handled it is rather advisable to pass by the tab "Print".

Roles

The administrators can export all the certificates corresponding to a given reference table.
The teachers with editing right can export only students' certificates of the course.

Formats of export of certificates

  • XML is a textual data format representing the information as a tag hierarchies and nested data. It allows the backup and restore of skills repositories, activities and certificates.
  • The XHTML format is an export-type format intended for Web page display.

  • The CSV is a textual format of tabular data (like in a spreadsheet).
    The CSV uses a separator, in general ';' between the cells of a table.

    The CSV is not really suited for nested data, as in a skills repository.
