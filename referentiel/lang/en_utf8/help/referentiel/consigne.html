Consignes

Les documents de consigne attachés à une tâche sont destinés à fournir des exemples ou des éléments susceptibles d'aider à remplir celle-ci.
A chaque tâche vous pouvez associer un ou plusieurs documents, soit en recopiant son adresse Web (URL), soit en déposant un fichier dans l'espace Moodle du cours.

Un document de consigne est constitué de :

  • Une description du document : Une courte notice d'information.
  • Le type de document : Texte, Exe, PDF, ZIP, Image, Audio, Video, etc. destiné à renseigner sur le format d'affichage.
  • Un lien
    • soit vers une URL : L'adresse Web du document (qui peut être un document déposé par vos soins dans l'espace Moodle).
        • Vous pouvez alors indiquer :
      • Un titre ou une étiquette
      • La fenêtre cible où s'ouvrira le document
    • soit vers un fichier déposé : Téléchargez un fichier depuis votre poste de travail.

Hints

Documents are linked with tasks to provide hints.
One or several documents may be linked with each task.

  • Document description: A short text.
  • Document type: Text, Exe, PDF, ZIP, Image, Audio, Video, etc., to support the display.
  • Link
    • or URL: Copy the Web address of the document (it can be a document uploaded in the Moodle site).
        • You can tape in :
      • A title
      • Target window for that document
    • or Uploaded file: Upload a file from you personnal computer
    • or Selected file: Select a file from the current course
