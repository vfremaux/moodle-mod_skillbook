Profiles

According to the type of skills repository, various information must be available to issue certificates.

Save them in the student’s and institution’s files.

Institutions

An institution file contains:

  • The institution ID number;
  • The institution name;
  • The institution’s reference;
  • The institution’s address;

Only teachers with editing rights can modify the institution’s files.

Students

A student’s file contains:

  • The student’s ID number;
  • Postcode and place of birth;
  • The postal address for correspondence;
  • The institution’s reference;

Students can modify and update their own file.
