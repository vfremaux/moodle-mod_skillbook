Export a Skills Repository

By exporting a repository in XML format you produce a backup that you can restore in your Moodle course with the reciprocal function Import.

By exporting a repository in XHTML format you produce a file that you can view in a Web page.

The files exported by the module repository are placed in the data folder "\referentiel" of the corresponding course (to be found in the \moodledata directory of the Moodle site).
